# E-Commerce REST API

A simple **E-Commerce REST API** built with **Node.js**, **Express**, and **MongoDB**.  
This API provides basic CRUD operations for products and serves as a foundation for building a full e-commerce platform.

---

## **Features**

- Create, read, update, and delete products (CRUD operations)
- RESTful API design
- Modular folder structure (controllers, routes, models)
- Environment variables for sensitive data
- Easy to extend with authentication, orders, and users

---

## **Tech Stack**

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB, Mongoose  
- **Development Tools:** Nodemon, dotenv
